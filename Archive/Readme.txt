Cara konfigurasi environtment program
- by kelompok 2, (FAJAR, ATHALLA, FARRISH)

1. Buat database: ticketingevent (mysql/phpmyadmin)
2. Import file .sql yang tersedia
3. Ekstrak program (EventTS)
4. Buka menggunakan IDE Java, nyalakan internet (Saran : Intellij IDEA)
5. Program akan mendownload beberapa library secara otomatis 
(karena ada interaksi dengan database dan ada .jar files)
6. Buka terminal, dan jalankan perintah mvn start atau bisa langsung menggunakan tombol start pada IDE
7. Program akan berjalan otomatis di localhost:8080
